Project Runesword

This is my LED-Bikehelmet project.
This project have been made as a mini-project at Aalborg University.
The bikehelmet is equipped with a LED-strip, a Micro:bit, a batterypack and some wires. Along with the programming of the bikehelmet, a wristband with another Micro:bit was added to it.
The programming of the Micro:bit did so when the helmet was tipped to either side, the LED's would light up, indicating which way the cyclist would go.
The Micro:bit on the wristband was programmed to give a signal to the helmet if the user raised their hand to stop. This would then make the LED's light up with a break light. Click on the image to see it in more detail.
